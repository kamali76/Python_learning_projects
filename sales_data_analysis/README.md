# Sales Data Analysis & Visualization using Python

## Project Description
This project analyzes retail sales data to understand customer purchasing patterns and sales trends.
The goal is to clean raw transaction data, perform analysis, and visualize key business insights using Python.

## Dataset
The dataset is sourced from Kaggle (Customer Shopping Dataset) and contains retail transaction records including product categories, customer demographics, payment methods, and purchase dates.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

## What I Did
- Loaded and explored raw sales data using Pandas
- Cleaned data by removing duplicates and fixing date formats
- Created a total sales metric for analysis
- Analyzed monthly sales trends and top product categories
- Visualized insights using line and bar charts

## Output
The project generates visualizations such as:
- Monthly sales trend chart
- Top product categories by revenue

All charts are saved in the `outputs/charts` folder.

## How to Run
1. Install dependencies from `requirements.txt`
2. Run `main.py`
3. View generated charts in `outputs/charts`
